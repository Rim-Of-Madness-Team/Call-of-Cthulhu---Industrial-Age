========================
The initial release did not have the proper wax recipe available. Wax can now be properly created at the Electric Stove, Fueled Stove, or Wood Stove by using meat.

Added many new lightsources. Now uses latest JecsTools code so that most added furniture can be safely placed on top of tables or other surfaces.
